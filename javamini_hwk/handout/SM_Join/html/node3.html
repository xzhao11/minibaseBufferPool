What You Have to Implement

The sortMerge constructor joins two relations R and S, represented by the Iterator object am1 and am2, respectively, using the sort-merge join algorithm.

Read the TupleUtils class carefully. You will need to call the setup_op_tuple method to set up joined tuple. You will also need to call the compare functions to compare the join columns of two tuples.
You will call the Sort constructor to sort a relation (represented by Iterator class).
You will also have to implement the get_next() method that returns the next joined tuple.
Note that all the relations are not saved in a heapfile or written out to a heapfile. Instead, you should use the functions provided by IoBuf class to operate tuples through a buffer.
