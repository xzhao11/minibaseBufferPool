Available Documentation

You should begin by reading the chapter on Implementation of Relational Operations, in particular, the section on Sort-Merge Join.

The package that contains SortMerge is iterator. You should also read the java documentation carefully for the package. You may find them at : http://path. Although there are many classes under the iterator class, you most likely only need a handful of them. However, you are encouraged to go over the rest of the documentation to gain an understanding of how things are being done. It will also help you to get started with your implementation.
